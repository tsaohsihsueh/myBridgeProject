# Key Features

Custom Hand Generation: Take control of your practice. You can specify exactly which opening bid you want to practice and set your partner’s point range to simulate specific bidding scenarios.

Instant Evaluation:

    Par Score Calculation: Automatically see the "Par Score" for any given hand, helping you understand the optimal result for both sides.

    Hand Analysis: Get an immediate evaluation of single hands to check your counting and logic.

BBO Integration: Generate a set of 8 random hands exported as a .lin file. This file is ready to be uploaded directly to Bridge Base Online (BBO) for practice matches or teaching sessions.
